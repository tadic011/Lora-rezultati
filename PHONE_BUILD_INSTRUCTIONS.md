# Kako da napraviš APK samo sa telefona

1. Otvori github.com u Chrome-u i prijavi se ili napravi nalog.
2. Napravi novi repository, npr. `lora-aplikacija`.
3. Otpremi sve fajlove iz ovog foldera u repository.
4. Otvori tab **Actions**.
5. Izaberi **Build Android APK**.
6. Klikni **Run workflow**.
7. Kada se build završi, otvori završeni workflow i preuzmi artifact **Lora-APK**.
8. Raspakuj ZIP koji GitHub skine i instaliraj `.apk` fajl na telefonu.

Ako telefon pita za dozvolu, uključi **Install unknown apps** za Chrome ili Files aplikaciju.
