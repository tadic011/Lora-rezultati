# Lora Score

Final Android project for tracking results in the card game Lora.

## Build APK on GitHub

1. Upload the contents of this folder to the root of the GitHub repository.
2. Open Actions.
3. Run Build Android APK.
4. Download artifact Lora-Score-APK.

## Rules implemented

- 4 players
- 4 rounds
- 7 games: Plus, Minus, Pop Herc, Dame, Herčevi, Lora, Igra po želji
- Positive and negative points
- Winner is the player with the lowest total score
- Total score updates automatically
- Local saving on phone
- Match history
