# Lora Score

Android aplikacija za praćenje rezultata u kartaškoj igri Lora.

## Pravila u aplikaciji
- 4 igrača
- 4 partije / kruga
- Igre: Plus, Minus, Pop Herc, Dame, Herčevi, Lora, Igra po želji
- Total Score se automatski računa
- Pobednik je igrač sa najmanjim brojem poena
- Poeni mogu biti pozitivni, negativni ili nula

## GitHub build
Uploadujte sadržaj ovog foldera u root repozitorijuma i pokrenite:
Actions -> Build Android APK -> Run workflow

APK će biti u artifactu: Lora-Score-APK
